COR24 Assembly Emulator - Project Description
=============================================

This is an educational browser-based emulator for the COR24 (C-Oriented RISC
24-bit) assembly architecture, written in Rust and compiled to WebAssembly.

COR24 Architecture Summary
--------------------------

- 24-bit RISC architecture designed for C language support
- 3 general-purpose registers (r0, r1, r2, 24-bit each)
- 5 special-purpose registers:
  - r3 = fp (frame pointer)
  - r4 = sp (stack pointer)
  - r5 = z (zero, for compare instructions)
  - r6 = iv (interrupt vector)
  - r7 = ir (interrupt return)
- Single condition flag (C) set by compare instructions
- 16MB byte-addressable memory (24-bit, little-endian)
- Variable-length instructions (1, 2, or 4 bytes)
- 211 valid instruction encodings

Unique COR24 Features:
- mov ra,c - Move condition flag directly to register (branchless comparisons)
- sxt/zxt - Sign/zero extension for byte-to-word conversion
- lc/lcu - Signed/unsigned 8-bit constant loading
- la - 24-bit address loading
- C-oriented register aliases for standard calling conventions
